# Security Policy

## Supported Versions

Currently, the `fusionaize-nexus-labs` infrastructure templates are rolling releases. Please ensure you are running the latest commit of the repository on your Core and Edge nodes.

## Reporting a Vulnerability

Security is a primary design principle of the Nexus architecture. If you discover a security vulnerability—whether in the Edge ingress proxy configs, the Core isolation layer, or the Workbench plugins—please do not report it via public issues.

Instead, please send an email to the repository owner directly. 

We process all reports promptly and will work with you to test and deploy a mitigation across the architectural reference guides.
